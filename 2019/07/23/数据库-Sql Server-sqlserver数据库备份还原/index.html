SqlServer 备份还原

发表于2019-07-23|更新于2020-05-22|数据库SqlServer
|阅读量:

SqlServer 备份

  1. 右键数据库，选择任务–>备份

  2. 在常规选项中，选择要备份的文件路径及名称

    • 清空原先备份的文件路径
    • 点击添加，选择磁盘文件目录
    • 可先选中一个备份文件，然后再修改文件名

  3. 确定后会添加到图中选中部分，点击确定，备份完成。

    SqlServer还原

    1. 将还原的.bak文件拷贝至服务器

    2. 右键要还原的数据库，选择任务–>还原–>数据库

    3. 左边选择常规，源选择设备，添加文件，选择刚才拷贝的备份文件

    4. 如果已经存在数据库，直接左边选择选项，勾选覆盖现有数据库，确定即可。
